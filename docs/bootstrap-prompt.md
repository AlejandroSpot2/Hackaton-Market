# Bootstrap Prompt

This repo was scaffolded from a prompt that asked for:
- Next.js plus TypeScript in `apps/web`
- FastAPI plus Pydantic in `services/api`
- lightweight Prefect-oriented orchestration
- React Flow atlas rendering
- local JSON storage for runs and fixtures
- polling-based progressive results
- three demo fixture scenarios for safe local demos

The resulting scaffold favors a stable end-to-end happy path over provider integrations.
